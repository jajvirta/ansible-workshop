Vagrant + VirtualBox + Ansible = <3

Tavoite

  • automatisoidaan "palvelinsoftan" asennus
  • automatisoidaan "sovelluksen" asennus
  • automatisoidaan sovelluksen smoketestaus

Valmistelut

... pitäisi olla tehtynä. Kokeile! 

            
C:\> vagrant up 
             
... 
C:\> vagrant ssh

Vagrant in a nutshell

  • polkaisee pystyyn virtuaalikoneita
  • monipuolisesti konffattavissa
  • peruskonffi on kiva pieni tekstifile

Ansible in a nutshell

  • "konfigurointijärjestelmä"
    (oikeasti: vähän niinku turbobuustattu shelliskriptaus)
  • ei agentteja
  • transparentisti ssh:n yli:
    master -> ssh (koodi + konffi) -> kohdekone:/tmp + magia
  • kannustaa tekemään idempotentteja konffeja
  • kannustaa automatisoimaan kaiken
